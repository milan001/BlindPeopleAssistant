Vehicle Detection, Tracking and Warning

1. Go to folder VehicleDetTrak in terminal.
2. Save the test video file to this folder.
3. Use the following command to detect and track vehicles in video and give audio warning message.
	python3 Vehicle.py videoname

Text Detection, Recognition and Speech

1. Go to folder TextDetRec in terminal.
2. Save the test image to this folder.
3. Use the following command to detect and recognize text in image and convert it to speech.
	python3 Text.py imagename
